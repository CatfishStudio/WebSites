$ Объявление переменных


ЧИСЛОВЫЕ ПЕРЕМЕННЫЕ

       $num = 10;

       $num = 12.54;

       $num = 0x3FF;

       $num = 5.97336e24;

===================


СТРОЧНЫЕ ПЕРЕМЕННЫЕ

       $Text = "текст";

       $Text = 'текст';

Вывод значения переменной:

       $T = "В переменной $Text";

Вывод имени переменной:

       $T = 'Имя переменной $Text';

===================


СЛУЖЕБНЫЕ СИМВОЛЫ

       \n новая строка

       \r возврат курсора

       \t горизонтальная табуляция

       \\ обратная касая черта

       \$ знак доллара

       \" кавычки

       \[0-7]{1,3} восьмеричная запись числа

       \x[0-9A-Fa-f]{1,2} шестнадцатеричная запись числа

===================
